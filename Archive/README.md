# KANgraphautoencoder
KAN-based graph autoencoder 

When running:
  1. data_processing.py, make sure to change the dataset, train/test/valid split, and batch size in the code to get desired objects
  2. any of the other architecture files, again be sure to match the input data with the correct parameters that you used in data_processing.py

Currently only have the MoleculeNet datasets

Code is adapted from Li et. al. - https://github.com/LongLee220/KA-GNN

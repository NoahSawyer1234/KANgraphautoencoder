# KANgraphautoencoder
KAN-based graph autoencoder 

Architectures are present in their respective files.
Processed datasets follow naming convention of '{Dataset}_{batch_size}.pth'. THEY DO NOT HAVE LABELS FOR SPLIT RATIO, AND ATTEMPTING TO CREATE A NEW PROCESSED DATASET WITH AN EXISTING BATCH SIZE WILL NOT OVERWRITE.
Models of focus are GAEs and GCNs. Topology models were for proof of concept/testing purposes
